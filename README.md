# Forensic Face Sketch Recognition System

A Java-based system that matches forensic face sketches with real images using AWS Rekognition.

## 🚀 Features
- Face sketch to real image matching
- AWS Rekognition integration
- Face collection search
- Image indexing

## 🛠 Tech Stack
- Java
- Maven
- AWS Rekognition
- AWS S3

## ⚙️ Setup Instructions
1. Clone the repo
2. Set AWS credentials as environment variables:
   - AWS_ACCESS_KEY
   - AWS_SECRET_KEY
3. Run the project using Maven

## 📌 Use Case
Helps law enforcement identify suspects using sketch-based inputs.
